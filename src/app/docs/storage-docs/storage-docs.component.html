Storage

.createDocument(

param1, param2

)

param1:

the name of your storage, e.g. storages.[name].

param1:

the name of a context variable that returns a storage object, e.g. context.[variable].

param2:

the name of an object function, e.g. objects.[name].

param2:

the name of a context variable that returns an object, e.g. context.[variable].

returns:

the inserted document.

returns:

null if the document does not pass the storage schema.

info:

Use this app to add an object (document) into storage.
