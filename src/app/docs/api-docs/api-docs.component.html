.status(

statusCode

)

statusCode:

the status code as a number

returns:

the status code

returns:

null if the argument is not a valid status code

info:

This app sets the request.statusCode property. Request.statusCode will not be set if the argument is not a number.

API

.send(

responseBody?

)

responseValue:

a string or boolean

responseBody:

an object function or context variable that returns an object function

returns:

the responseValue or responseBody

returns:

an empty object if there is no value, or if the value is not a string, boolean, or JSON parseable object.

info:

This app ends the request workflow. The responseValue is set in a standard JSON response body (message: value), and the responseBody is set as the response body. Any rows after API.send() are not processed.
