API

.status(

param1

)

param1:

the status code as a number, e.g. number.[statusCode].

returns:

the status code as a number.

returns:

null if the number is not a valid status code.

info:

This app sets the Express request.statusCode property. The request.statusCode property will not be set if the parameter is not a valid status code.

API

.send(

param1

)

param1:

a string or boolean, e.g. string.[value] or boolean.[value].

param1:

the name of an object function, e.g. objects.[name].

param1:

the name of a context variable that returns an object, e.g. context.[variable].

returns:

the Express response body as message:value (if a string or boolean), or the entire object if an object.

returns:

an empty object if there is no value, or if the value is not a string, boolean, or JSON parseable object.

info:

This app ends the request workflow. Any rows after API.send() are not processed.
