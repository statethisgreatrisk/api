Object

.get(

param1

)

param1:

the name of an object function, e.g. objects.[name].

returns:

the object that is the result of running the object function.

returns:

null if no object or if there is an error calling the object function.

Object

.getProperty(

param1, param2

)

param1:

the name of an object function, e.g. objects.[name].

param1:

the name of a context variable that returns an object, e.g. context.[variable].

param2:

the field to get from the object as a string, e.g. string.[field].

returns:

the value of the field.

returns:

null if no object, if no value or if there is an error calling the object function.
