Functions must begin with function fn(context) { and end with }.


The function can return any value and may be rejected for using certain restricted words within the function body. Callback functions can also be created for forEach, map, filter and reduce functions.


ForEach, Map and Filter functions must begin with function fn(element, index, array, context) { and end with }.


Reduce functions must begin with function fn(accumulator, value, index, array, context) { and end with }.
