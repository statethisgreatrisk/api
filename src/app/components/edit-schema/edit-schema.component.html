Schema name can have up to 80 characters, may only contain alphanumeric characters and underscores (_), is case sensitive and must be unique.

Schemas are based on Zod 3.x. See the Zod documentation for details.


Schema functions must begin with function fn(zod) { and end with }.


Schemas are versioned and documents will only be shown in the Document Viewer if they pass the latest version of the schema. Saving the schema will always create a new version.


The schema function should return a zod schema and may be rejected for using certain restricted words within the function body.
