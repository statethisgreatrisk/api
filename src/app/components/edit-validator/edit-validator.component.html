Field selection and validators are based on express-validator 7.x. See the express-validator documentation for details.


Validator functions must begin with function fn(check, body, cookie, header, param, query, schemas, ObjectId) { and end with }.


The validator function should return an express-validator validator or sanitizer and may be rejected for using certain restricted words within the function body.
